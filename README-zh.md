<p align="center">Vae-UI - 一个风格简约的 Vue3 UI 组件库</p>

- 💪 Vue 3.0 Composition API
- 🔥 Written in TypeScript

该项目仍在紧张开发中。欢迎加入我们，请提交 `pull request`。

### 预览

请在命令行输入以下命令

```bash
$ yarn website-dev
```

这样你就能预览到所有组件。

### UI 设计稿

使用的是来自 [rsuite](https://rsuitejs.com/design/default/#artboard1) 的设计稿，在此特意感谢。
